# Penguin Species Classification using Machine Learning

## 📌 Overview
This project uses the Kaggle Penguin dataset to perform both supervised and unsupervised machine learning. It classifies penguin species and also explores hidden patterns using clustering techniques.

---

## 📊 Dataset
- Source: Kaggle Penguin Dataset
- Features include: bill length, flipper length, body mass, sex, and island

---

## ⚙️ Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## 🧪 Workflow

### 1. Data Preprocessing
- Removed missing values
- Applied one-hot encoding for categorical variables
- Standardized features using StandardScaler

---

### 2. Supervised Learning Models
- Logistic Regression
- Random Forest Classifier

Both models were trained to classify penguin species.

---

### 3. Model Evaluation
- Accuracy Score
- Confusion Matrix

---

### 4. Unsupervised Learning
- Agglomerative Hierarchical Clustering
- Grouped penguins into clusters without labels
- Visualized clusters using scatter plots

---

## 📈 Results
- Compared performance of Logistic Regression and Random Forest
- Random Forest showed better classification performance
- Clustering revealed natural grouping patterns in dataset

---

## 📌 Key Insights
- Flipper length and bill length are strong indicators of species
- Supervised models perform well on this dataset
- Clustering provides meaningful unsupervised grouping
